## Project Overview
- A **priority encoder** is a combinational circuit that converts multiple input lines into a smaller number of output lines based on priority.
- Only the **highest-priority active input** is encoded into the output at a time.
- Common types include **octal-to-binary**, **decimal-to-BCD**, and **keyboard encoders**.
